### 1. Аутентификация

#### 1.1 Регистрация пользователя

- **Метод**: `POST`
- **URL**: `/api/auth/registration`
- **Тело запроса**:
```json
{
"login": "string",
"password": "string"
}
```
- **Ответ**:
- **Успех**: `201 Created` с сообщением о успешной регистрации.
- **Ошибка**: `400 Bad Request` при некорректных данных.

#### 1.2 Вход пользователя

- **Метод**: `POST`
- **URL**: `/api/auth/login`
- **Тело запроса**:
```json
{
"login": "string",
"password": "string"
}
```
- **Ответ**:
- **Успех**: `200 OK` с JWT токеном.
- **Ошибка**: `401 Unauthorized` при неверных данных.

#### 1.3 Получение текущего пользователя

- **Метод**: `GET`
- **URL**: `/api/auth`
- **Заголовок**: `Authorization: Bearer {token}`
- **Ответ**:
- **Успех**: `200 OK` с данными текущего пользователя.
- **Ошибка**: `401 Unauthorized` при отсутствии или некорректном токене.

### 2. Управление пользователями

#### 2.1 Получение всех пользователей

- **Метод**: `GET`
- **URL**: `/api/user`
- **Заголовок**: `Authorization: Bearer {token}`
- **Ответ**:
- **Успех**: `200 OK` с массивом пользователей.
- **Ошибка**: `403 Forbidden` при отсутствии прав доступа.

#### 2.2 Получение пользователя по ID

- **Метод**: `GET`
- **URL**: `/api/user/{id}`
- **Заголовок**: `Authorization: Bearer {token}`
- **Ответ**:
- **Успех**: `200 OK` с данными пользователя.
- **Ошибка**: `404 Not Found` при отсутствии пользователя.

#### 2.3 Удаление пользователя

- **Метод**: `DELETE`
- **URL**: `/api/user/{id}`
- **Заголовок**: `Authorization: Bearer {token}`
- **Ответ**:
- **Успех**: `204 No Content`.
- **Ошибка**: `403 Forbidden` при отсутствии прав доступа.

## Документация API

### Общая информация

Этот API предоставляет функциональность для управления пользователями, ролями, группами, заметками и расписанием. Он использует JWT для аутентификации и авторизации пользователей.

### Аутентификация

Для доступа к защищенным маршрутам необходимо использовать JWT токен в заголовке `Authorization: Bearer {token}`.

### Формат данных

Все запросы и ответы используют формат JSON.
